# OOPs in C++

This repository contains C++ programs demonstrating core Object-Oriented Programming (OOPs) concepts.

---

## 🚀 Concepts Covered

- ✅ Classes & Objects  
- ✅ Constructors and Destructors  
- ✅ Inheritance (Single, Multilevel, etc.)  
- ✅ Polymorphism (Overloading & Overriding)  
- ✅ Encapsulation  
- ✅ Abstraction  
- ✅ File Handling in C++  
- ✅ Exception Handling  

---

## 📁 Folder Structure

Each `.cpp` file contains examples and explanations related to one OOP concept, such as:
- `inheritance.cpp`
- `virtual_functions.cpp`
- `file_handling.cpp`
- `exception_handling.cpp`
- and more...

---

## 🛠 How to Run

Make sure you have a C++ compiler installed like `g++` (comes with MinGW or Linux).  
To compile and run any file:

```bash
g++ filename.cpp -o output
./output
